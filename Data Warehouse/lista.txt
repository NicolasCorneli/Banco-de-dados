Explicação: Data warehouses são projetados para atender às necessidades de relatórios e análises de uma organização, abrangendo diferentes níveis hierárquicos, enquanto bancos de dados transacionais (OLTP) focam em operações diárias de processamento de transações.

Resposta: C) proporcionar um ambiente que permita realizar análise dos negócios de uma empresa com base nos dados por ela armazenados

Explicação: Data warehouses são projetados para análise de dados históricos e suporte à tomada de decisões, fornecendo uma visão consolidada dos negócios.

Explicação: A fase de transformação no processo de ETL envolve modificar e adequar os dados ao modelo do data warehouse, garantindo que estejam prontos para análise.
